about the searchable periodic table

how to search

You can search by element name, symbol, atomic number, category, property, or discovery date.

A couple of things to know:

  • When you start typing, everything will be dimmed or (for mobile users) disappear. Don't panic. This is normal. Things re-appear when you delete your query or it finds something that matches the query.
  • It's not case sensitive. You can mix-and-match capital and lower case letters as much as you want. If you don't see anything for 'eh', trying 'Eh' won't help.
To search by symbol, enter one or two letters. So, 'hy' returns nothing because it's not a chemical symbol for any element.

To search by name, start typing the element name you're after. You can't search by more than one name at a time "carbon" will work, but "carbon and oxygen" won't.

To search by discovery date, the query needs to be in the form "discovered/found before/after/in/during year". You can find all elements that were known about before 1840 by searching "discovered before 1840" OR "found before 1840". To find elements that were found in 1840, type "found in 1840" OR "discovered during 1840" (There weren't any found that year. Everyone was too busy signing the treaty or something). To find elements that have been found since 1840, the query needs to be "discovered after 1840" OR "found after 1840".

Searching by atomic number is easy. Just type "104".

To search by category or property, just type the category or property. To see all the radioactive elements, search "radioactive". To see the halogens, type "halogen" or "halogens". Alternatively, you can search "group 17", "grp17". In a similar vein "row 3" or "period 3" also works. Also, "s-block" or "s block" or "sblock" will show you all the s-block elements.

other features

  • you can control the sliders by using the arrow keys
  • you can go between element modals using the arrow keys
  • you can close the element modal by pressing the esc key
  • clicking the celsius button changes the temperature units

why does this exist?

This project was a product of stupidity. I learnt that web pages have a property called "tables" and thought "y'know, the periodic table is a table. I could make one for a website". So I did. But it was using the wrong tool. It was like drawing the periodic table in Excel.

Github user Mirlington pointed out to me that this should be using a grid, not a table. He pointed out many other dumb design decisions I made along the way. This project would be no where near as good without his help as a consultant.

The quiz was initially developed as a set of cards produced by PIPERS. The original plan was for these cards to be sent to high schools. But, because the internet exists, all the questions and images were translated into webpages. The quiz is now infintely scalable.
